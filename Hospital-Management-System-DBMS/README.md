# Hospital Management System (DBMS Project)

## ?? Description

This project is a desktop-based Hospital Management System developed using Python Tkinter and MySQL.

It allows management of:

* Patients
* Doctors
* Appointments
* Beds
* Emergency Cases

## ?? Technologies Used

* Python
* Tkinter
* MySQL

## ?? Features

* Add/Delete Patients
* Manage Doctors
* Book & Update Appointments
* Bed Management System
* Emergency Handling

## ??? Database Setup

1. Open MySQL
2. Run the `database.sql` file
3. Update your MySQL password in `main.py`

## ?? How to Run

1. Install Python
2. Install mysql connector:
   pip install mysql-connector-python
3. Run:
   python main.py
